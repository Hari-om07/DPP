# DPP
Diamond Price Prediction using Machine learning
Developed Machine learning model for the Predicting the price of Diamonds.
Used features like carat, color and clarity for training our model and based on that prediction is made.
Different classifiers like linearRegressor, KNeighborsClassifier, DecisionTreeRegressor has been trained but the RandomforestRegressor outperforms very well in prediction.
The model has been deployed on a web page using flask.
